Qu'est ce que PHP? 
    Php = Hypertext Preprocessor : c'est un langage de scripts généraliste et Open Source, spécialement conçu pour le développement d'applications web dynamiques. Il peut être intégré facilement au HTML.

Les sites internet modernes sont généralement utilisés sur deux machines/serveurs/ordinateur qui vont intéragirent ensemble pour le fonctionnement optimal du site.

Le navigateur ne comprend que 3 langages :
    > HTML
    > CSS
    > JAVASCRIPT
